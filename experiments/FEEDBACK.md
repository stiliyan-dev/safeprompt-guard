# FEEDBACK

## Internal pilot comments

- “This is useful the second I paste a token by mistake.”
- “I want a one-click redact option, not just cancel.”
- “The warning should tell me whether it found an email, key, or card before I open the modal.”
- “I would use this on ChatGPT and Claude if it stayed local.”

## Pain points

- Heuristic send detection can feel inconsistent.
- Warning placement is generic rather than attached to the exact field.

## Feature requests

- Redact instead of only warn.
- Site-specific controls for trusted domains.
- Shared team secret patterns.

## Complaints

- No attachment scanning.
- No history of blocked prompts.
